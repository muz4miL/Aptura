# Aptura Tech Solutions

Official website for **Aptura Tech Solutions** - your Digital Innovation Partner specializing in Software, AI and Web Solutions.

---

## Tech Stack

- **Next.js 16** (App Router) - Server-side rendering and SEO-optimized pages
- **Tailwind CSS v4** - Utility-first styling with PostCSS integration
- **Framer Motion and GSAP** - Smooth animations and scroll effects
- **React Icons and Lucide** - Icon library
- **EmailJS** - Contact form integration

---

## Getting Started

npm install
npm run dev

Open http://localhost:3000 in your browser.

## Build

npm run build
npm start

---

## Project Structure

app/              - Next.js App Router pages
components/       - Reusable UI components
public/           - Static assets (images, icons, videos)
